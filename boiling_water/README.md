# Heat Transfer Explorer

## Folder structure

```text
heat-transfer-github/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── app.js
└── videos/
    ├── 01_no_activity.mp4
    ├── 02_low_intensity.mp4
    ├── 03_med_low_intensity.mp4
    ├── 04_med_high_intensity.mp4
    └── 05_high_intensity.mp4
```

Copy your five MP4 files into the `videos` folder before uploading the project to GitHub.

## GitHub Pages

1. Create a new GitHub repository.
2. Upload the full folder structure.
3. Open **Settings**.
4. Select **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Choose the `main` branch and `/root`.
7. Save.
8. GitHub will provide the published URL.

## Required video format

Use the same settings for all five videos:

- MP4
- H.264 video
- No audio, or muted audio
- 1920 × 1080
- Exactly 3 seconds
- Matching first and last frame
- Web optimized / Fast Start enabled
